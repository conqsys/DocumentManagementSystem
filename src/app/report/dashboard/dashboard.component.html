Document Report

This report enables you to perform a wild card search for documents on all of the index fields as well as the comments field and can be narrowed down to search a specific date range.


User Report

This report shows activities for each user within eBridge and contains the date of the activity, username, address, activity type, all of the index fields, document type, number of pages, and size for the document.


Pages Report

This report displays the total number of pages added to the system by each user for a specific date range and the percentage of the total page count for that date range.


Edited Documents Report

This report displays a list of all of the documents that have been edited by each user for a specific date range. It contains the date edited and the value of each index field for the given document. It can be filtered by date, and by user.


Deleted Documents Report

This report displays a list of all of the documents that have been deleted by each user for a specific date range. It contains the date deleted and the value of each index field for the given document. It can be filtered by date, and by user.


Workflow Report

This report displays a list of all of the documents that have been routed through the workflow process by each user for a specific date range. It can be filtered by date, by status, and by user.


Admin Report

This report displays a list of the users and their permissions within eBridge. It can be filtered by user name and the date the user last logged in.
